# Credit Card Fraud Detection Using Machine Learning

## CodeSoft Data Science Internship Project

### Project Overview

This project focuses on detecting fraudulent credit card transactions using Machine Learning techniques. The dataset was preprocessed, balanced using SMOTE, and used to train a Logistic Regression model for fraud detection.

### Objective

To develop a machine learning model that can identify whether a credit card transaction is fraudulent or genuine.

### Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Scikit-learn
* Imbalanced-learn (SMOTE)

### Dataset Information

The dataset contains credit card transaction records with the following classes:

* Class 0 → Genuine Transaction
* Class 1 → Fraudulent Transaction

### Project Workflow

1. Data Collection and Loading
2. Data Preprocessing
3. Data Normalization
4. Handling Class Imbalance using SMOTE
5. Train-Test Split
6. Model Training using Logistic Regression
7. Model Evaluation
8. Performance Analysis

### Model Used

* Logistic Regression

### Performance Metrics

| Metric    | Score     |
| --------- | --------- |
| Accuracy  | 98%       |
| Precision | 98% - 99% |
| Recall    | 98% - 99% |
| F1-Score  | 98%       |

### Results

The model achieved excellent performance in detecting fraudulent transactions with high precision, recall, and F1-score values. SMOTE helped balance the dataset and improved fraud detection capability.

### Conclusion

A machine learning-based fraud detection system was successfully developed. The model effectively classifies credit card transactions as fraudulent or genuine and achieved an overall accuracy of 98%.
