# Iris Flower Classification using Machine Learning

## Project Overview

The Iris Flower Classification project is a machine learning classification system designed to identify the species of Iris flowers based on their physical measurements. The project uses the famous Iris dataset, which contains information about sepal length, sepal width, petal length, and petal width for three Iris species: Setosa, Versicolor, and Virginica.

## Objective

The objective of this project is to build a machine learning model that can accurately classify Iris flowers into their respective species using supervised learning techniques.

## Dataset Description

The Iris dataset contains the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target Classes:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## Methodology

### Data Collection

The Iris dataset was loaded and analyzed to understand its structure and characteristics.

### Data Preprocessing

* Checked for missing values
* Performed data cleaning
* Prepared features and target variables
* Split the dataset into training and testing sets

### Model Building

A Decision Tree Classification algorithm was used to train the model using the training dataset.

### Model Evaluation

The model performance was evaluated using:

* Accuracy Score
* Confusion Matrix
* Classification Report

## Results

The model successfully classified Iris flowers into their respective species with high accuracy.

**Model Accuracy:** 95%–100%

The results demonstrate that the selected flower measurements are highly effective in distinguishing between the three Iris species.
