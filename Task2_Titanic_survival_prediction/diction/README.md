# Titanic Survival Prediction

## Project Overview

The Titanic Survival Prediction project is a Machine Learning classification project that predicts whether a passenger survived the Titanic disaster. The model uses passenger information such as age, gender, passenger class, fare, and family details to make predictions.

## Objective

The main objective of this project is to build a predictive model that can classify passengers as survived or not survived based on historical Titanic passenger data.

## Dataset

The dataset contains passenger information including:

* Passenger Class (Pclass)
* Gender (Sex)
* Age
* Fare
* SibSp (Number of siblings/spouses aboard)
* Parch (Number of parents/children aboard)
* Embarked Port
* Survival Status (Target Variable)

## Data Preprocessing

The following preprocessing steps were performed:

* Handled missing values.
* Removed unnecessary columns.
* Encoded categorical variables into numerical values.
* Selected important features for model training.
* Split the dataset into training and testing sets.

## Machine Learning Model

Algorithm Used:

* Random Forest Classifier

Training and Testing Split:

* 80% Training Data
* 20% Testing Data

## Results

* Model Accuracy: 82%
* The model successfully predicts passenger survival with good performance.

## Key Findings

* Gender was one of the most important factors affecting survival.
* Female passengers had a higher survival rate than male passengers.
* Passenger class significantly influenced survival chances.
* Higher-class passengers had better survival rates.
* Random Forest provided reliable classification results.

## Conclusion

This project demonstrates how Machine Learning can be used to solve real-world classification problems. By applying data preprocessing, feature engineering, and the Random Forest algorithm, the model achieved an accuracy of 82%, providing valuable insights into passenger survival patterns during the Titanic disaster.
